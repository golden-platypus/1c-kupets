# 🏪 1С:Купец

> Учебная конфигурация на платформе **1С:Предприятие 8.3** для комплексной автоматизации торгового предприятия: управление торговлей, складом, кадрами, зарплатой и бухгалтерским учётом.

---

## 📌 О проекте

**«1С:Купец»** — многофункциональная учебная конфигурация, разработанная с нуля на платформе 1С:Предприятие 8.3.

Проект охватывает торговый и складской учёт, работу с контрагентами и договорами, кадровый учёт, расчёт заработной платы и отдельные механизмы бухгалтерского учёта. Конфигурация демонстрирует практическое применение различных объектов и механизмов платформы: справочников, документов, регистров, управляемых форм, СКД, ролей, общих модулей и ввода документов на основании.

## 🏗️ Архитектура конфигурации

### 📂 Подсистемы

Конфигурация разделена на 4 функциональных блока:

```
1С:Купец
├── 🛒 ТорговыйУчет        — продажи, закупки, склад, номенклатура
├── 📋 НСИ                 — нормативно-справочная информация
├── 👥 КадровыйУчет        — сотрудники, оклады, прогулы, зарплата
└── 📒 БухгалтерскийУчет   — план счетов, проводки
```

## 🗂️ Объекты конфигурации

### Справочники (7 шт.)

| Справочник | Назначение |
|---|---|
| `Номенклатура` | Товары и услуги компании |
| `Контрагенты` | Покупатели и поставщики |
| `Договоры` | Договоры с контрагентами |
| `Сотрудники` | Кадровый учёт |
| `СкладыКомпании` | Склады предприятия |
| `ВидыХарактеристик` | Виды характеристик номенклатуры |
| `ВариантыЗначенийХарактеристик` | Значения характеристик (цвет, размер и т.д.) |

---

### 📄 Документы (7 шт.)

| Документ | Подсистема | Назначение |
|---|---|---|
| `ЗаказКлиента` | Торговый учёт | Оформление заказа от покупателя |
| `РасходнаяНаклодная` | Торговый учёт | Продажа товаров со склада |
| `ПриходнаяНаклодная` | Торговый учёт | Поступление товаров на склад |
| `СчетНаОплату` | Торговый учёт | Выставление счёта контрагенту |
| `ПКО` | Финансы | Приходный кассовый ордер |
| `РасчетнаяВедомость` | Кадры | Расчёт и начисление зарплаты |
| `ФиксацияПрогула` | Кадры | Учёт нарушений рабочей дисциплины |

> 🔢 Складские накладные (приход/расход) используют общий **нумератор** `СкладскойНумератор` и объединяются в **журнал** `СкладскиеНаклодные`.

## 🔄 Логика работы

```text
📦 Приходная накладная
         │
         ▼
Регистр «ОстаткиНоменклатуры» (+ количество)
         │
         ▼
🛒 Заказ клиента
         │
         ├──► Счёт на оплату
         │
         └──► Расходная накладная
                    │
                    ├──► Регистр «Продажи»
                    └──► Регистр «ОстаткиНоменклатуры» (- количество)

💰 Приходный кассовый ордер
         │
         ▼
Регистр «БухгалтерскиеПроводки»
(учебный пример двойной записи)

📊 Отчёты:
Продажи · Остатки номенклатуры · ОСВ

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

👥 Сотрудники + Оклады сотрудников
         │
         ▼
📋 Расчётная ведомость
         │
         ▼
Регистр расчёта «Зарплата»
         │
         ▼
📊 Отчёт «Заработная плата»
```

## 📋 Требования для запуска

- **1С:Предприятие** версии 8.3.26 или выше
- Любая поддерживаемая СУБД (файловый режим, PostgreSQL, MS SQL и др.)

### Установка конфигурации

1. Создать новую информационную базу в конфигураторе
2. Загрузить: `Конфигурация → Загрузить конфигурацию из файла` → `Купец.cf`
3. Обновить конфигурацию базы данных (`F7`)

### Загрузка с тестовыми данными

1. Создать новую информационную базу
2. `Администрирование → Загрузить информационную базу` → `Купец.dt`
